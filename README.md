# HR Analytics вЂ“ Absenteeism Dashboard

## Project Overview

This project analyzes employee absenteeism using **MySQL** and **Power BI**. The objective is to identify absenteeism patterns, employee health indicators, seasonal trends, and factors that contribute to employee absence.

The dashboard provides HR teams with insights that can support attendance monitoring and workforce planning.

---

## Tools & Technologies

- MySQL
- SQL
- Power BI
- GitHub

---

## Dataset

The project uses three datasets:

- `absenteeism_at_work.csv`
- `compensation.csv`
- `reasons.csv`

---

## Project Workflow

1. Imported CSV datasets into MySQL.
2. Performed data analysis using SQL.
3. Joined multiple tables to create a reporting dataset.
4. Connected MySQL to Power BI.
5. Designed an interactive HR Analytics dashboard.

---

## SQL Analysis

The SQL queries include:

- Employee health analysis
- Non-smoker employee identification
- BMI categorization
- Season classification
- Employee absenteeism analysis
- Data preparation for Power BI dashboards

---

## Dashboard Features

- Total Employees
- Total Absenteeism Hours
- Average Absenteeism by Month
- Average Absenteeism by Day of Week
- Employee Distribution by:
  - BMI
  - Education
  - Number of Children
  - Pets
- Absenteeism Reasons
- Transportation Expense vs Workload Analysis
- Season Filter (Winter, Spring, Summer, Fall)
